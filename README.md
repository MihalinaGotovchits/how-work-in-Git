# Работа в Git основные команды


*pwd* - определение текущей дериктории


*cd ~* - перемещение в домашнюю директорию


*ls* - вывести содержимое директории (список файлов)


*ls ~* - вывести содержимое домашней директории


*"ls ..* - показать содержимое родительской директории


*ls -a* - вывести список файлов, в котором отображаются скрытые файлы 


*cd ..* - вернуться на одну папку назад


*cd .* - перейти в текущую директорию


*cd директория/директория* - переход через несколько директорий



#**Создание файлов и директорий**


*touch %ИМЯ_ФАЙЛА%* - создать новый файл


*mkdir %ИМЯ_ДИРЕКТОРИИ%* - создать новую директорию


*mkdir -d* - создать структуру директорий


#**Копирование файлов**


*cp что_копируем куда_копируем* - копировать файл


можно копировать сразу несколько файлов:


*cp что_копируем что_копируем что_копируем куда_копируем*


#**Перемещение файлов и папок**


*mv что_перемещаем куда_перемещаем* - переместить файл (удаляет в одном месте и перемещвет в другое место)


#**Чтение и удаление файлов**


*cat %ИМЯ_ФАЙЛА%* - прочитать файл в консоль (только для текстовых файлов)


*rm %ИМЯ_ФАЙЛА%* - удалить файл (без права на востановление)


*rmdir %ИМЯ_ПАПКИ%* - удалить папку (без права на востановление)


*rm -r %ИМЯ_ПАПКИ%* - удалить папку со всем ее содержимым (без права на востановление)



**Выполнить сразу несколько команд можно при помощи "&&" между командами**


**Для обращения к последней команде - нажать на стрелочку "вверх"**


#Автозаполнение


Ввести начало команды и два раза нажать Tab


*cd c:/* - переместиться в корневую директорию


#**Создание репозитория**


1. Переместиться в нужную папку

2. Ввести команду git.init

Если случайно создала репозиторий не в той папке:

1. Перейти в репозиторий

2. "rm -rf .git"


#**Состояние репозитория**

*git status* - текущее состояние репозитория


#**Добавление файлов в репозиторий**

1. Создать в репозитории файл
2. *git add %ИМЯ_ФАЙЛА%* - Добавить файл в репозиторий
3. *git add --all* - добавить все файлы в репозиторий
4. *git add .* - добавить всю текущую папку
5. *git commit -m 'КОММЕНТАРИЙ'"* - сохранение файлов с комментариями


#**Посмотреть историю коммитов**


*git log* - -история коммитов


№**Создание ключа**


*ssh-keygen -t ed25519 -C "ИМЯ_ПОЛЬЗОВАТЕЛЯ" "

*ls -a ~/.ssh* - показать содержимое папки .ssh

*ls -la ~/.ssh* - показать содержимое папки .ssh  в виде списка файлов


№**Общение между локальным и удаленным репозиториями**


*git remote add* - привязать удаленный репозиторий к локальному

*git remote -v* - проверка связанности репозиториев между собой

*git push* - отправить изменения на удаленный репозиторий

№**Клонирование репозиториев**

*git clone* - клонировать репозиторий (создастся копия удаленного репозитория)
